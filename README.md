# 내 직업들 
세계의 파멸을 막기 위해 바하무트의 비밀을 풀어나가는 "기사"

진정한 왕이 되기위해 분투하며 적과 맞서 싸우는 100만 자루의 엑스칼리버를 뽑아든 100만 명의 "아서" 중 한 사람

진수부에 착임해서 칸무스를 지휘하는 "제독"

S급 발키리들이 소속된 히페리온 함대의 "함장"

하슬라를 수호하는 챔피언 "용사 단장"

그리폰&크루거의 전술인형 부대를 지휘하는 "전술 지휘관"

광석병 감염자들을 구하기 위해 움직이는 로도스 아일랜드의 간부 " 닥터"

인류의 영웅들과 함께 이성의 신과 맞서 싸우는 칼데아 소속의 "마스터"

세이렌을 상대하는 아주르 레인의 "지휘관"

수 많은 아이돌들을 톱까지 성장시킨 " 프로듀서"

별의 섬 이스탈시아로 향하는 기공단의 "단장"

물질계를 위협하는 마왕과 신계의 음모에 맞서는 "왕자"

비밀리에 세계를 수호하며 역사를 기록하는 전승학파의 "관찰자"

철충과의 싸움에서 전멸한 인류의 마지막 생존자 "사령관"

우마무스메들을 육성하는 트레센 학원의 "트레이너"

차세대 아이돌 극장형 스튜디오 쓰리세븐의 "2대 지배인"

수많은 프린세스들과 랜드솔을 여행하는 "프린세스 나이트"

관리국에 정식으로 인증받은 태스크포스 코핀 컴퍼니를 운영하는 "사장"

여신 디체에게 권능을 나누어 받아 7번째 세계를 지키는 "계승자"

오로리안들과 함께 암귀에 맞서싸우는 최후의 "아이테르"

천리의 주관자에게 빼앗긴 혈육를 찾아 헤매는 "여행자"

학원도시 키보토스의 연방수사동아리「S.C.H.A.L.E」의 "선생님"
